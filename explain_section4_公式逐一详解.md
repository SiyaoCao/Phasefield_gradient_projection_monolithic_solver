## 4.1 Cyclic tension-compression test（循环拉压算例）

### 公式 4.1-1（Lamé 第一参数）

\[
\lambda = 121.15\ \mathrm{kN/mm^2}
\]

- 含义：\[\lambda\] 是线弹性各向同性材料的 Lamé 参数之一，控制体积变形刚度。
- 与体积模量关系：\[K = \lambda + \tfrac{2}{3}\mu\]。
- 在相场断裂中，\[\lambda\] 与 \[\mu\] 一起进入弹性能密度，进而影响驱动裂纹扩展的能量释放。

### 公式 4.1-2（Lamé 第二参数 / 剪切模量）

\[
\mu = 80.77\ \mathrm{kN/mm^2}
\]

- 含义：\[\mu\] 是剪切模量，主要控制偏应变能（形变）部分。
- 推导背景：在线弹性本构中，\[\boldsymbol{\sigma}=2\mu\boldsymbol{\varepsilon}+\lambda\,\mathrm{tr}(\boldsymbol{\varepsilon})\mathbf{I}\]。
- \[\mu\] 越大，材料对剪切变形越“硬”，裂纹路径和临界载荷会受影响。

### 公式 4.1-3（临界能量释放率）

\[
g_c = 2.7 \times 10^{-3}\ \mathrm{kN/mm}
\]

- 含义：\[g_c\] 是 Griffith 型断裂韧度（单位裂纹面积所需能量）。
- 在相场模型中，\[g_c\] 乘以裂纹表面密度泛函（见公式 4.1-11）得到裂纹耗散能。
- \[g_c\] 越大，裂纹越难扩展。

### 公式 4.1-4（退化函数中的小参数）

\[
k = 0.0
\]

- 含义：\[k\] 是退化函数中避免刚度完全退化时奇异性的“残余刚度”参数。
- 本文该算例取 \[k=0\]，这里是**模型设定中的精确零值**（不是数值容差阈值）；表示不引入人工残余刚度。这会使问题更“脆”，也更考验非线性求解器的稳健性。

### 公式 4.1-5（退化函数记号）

\[
g(d)
\]

- 含义：\[g(d)\] 将材料弹性能随损伤变量 \[d\] 退化。
- 常见形式（背景说明）为 \[g(d)=(1-d)^2+k\]；本文第4章中主要给出符号，不重复展开具体式子。
- 物理作用：\[d\to1\] 时拉伸承载能力退化，形成裂纹带。

### 公式 4.1-6（相场内长度尺度）

\[
l = 0.0075\ \mathrm{mm}
\]

- 含义：\[l\] 控制弥散裂纹带宽度，越小裂纹越尖锐，但网格要求更高。
- 数值意义：\[l\] 与网格尺寸 \[h\] 的比值决定裂纹带解析精度。

### 公式 4.1-7（局部网格尺寸）

\[
h = 0.00265\ \mathrm{mm}
\]

- 含义：\[h\] 是优选区域的单元尺寸。
- 与 \[l\] 配合用于检验是否有足够单元解析相场过渡区。

### 公式 4.1-8（网格-长度尺度比）

\[
h/l \approx 0.353 \approx 1/3
\]

- 含义：每个 \[l\] 尺度内约 3 个单元（甚至更多节点）来分辨损伤梯度。
- 必要推导：
  \[
  h/l = 0.00265/0.0075 \approx 0.353\approx 1/3
  \]
- 结论：该比值满足相场裂纹常见离散要求（至少 3 个单元覆盖主要过渡宽度）。

### 公式 4.1-9（加载阶段位移）

\[
u_y = 5.9 \times 10^{-3}\ \mathrm{mm}
\]

- 含义：上边界竖向位移达到该值时，裂纹接近失稳扩展临界前状态。

### 公式 4.1-10（失稳后位移）

\[
u_y = 6.0 \times 10^{-3}\ \mathrm{mm}
\]

- 含义：从 \[5.9\times10^{-3}\] 到 \[6.0\times10^{-3}\] 的微小增量触发快速扩展，体现脆性断裂“突发性”。

### 公式 4.1-11（相场变量取值范围）

\[
[0,1]
\]

- 含义：\[d=0\] 完好，\[d=1\] 完全断裂；数值上必须保持在该区间。
- 对应约束：\[d^{(n)}\le d \le 1\]（不可逆 + 上界）。

### 公式 4.1-12（完全开裂时间标记）

\[
t = 6.0 \times 10^{-3}
\]

- 含义：伪时间到此值后试件失去拉伸承载。

### 公式 4.1-13（反向压缩开始时间）

\[
t = 14.0 \times 10^{-3}
\]

- 含义：标记载荷路径进入压缩阶段，用于说明张压非对称响应。

### 公式 4.1-14（裂纹耗散能总式，文中式(33)）

\[
E_{\mathrm{crack}} = g_{\mathrm{c}}\Gamma_{l}(d) = g_{\mathrm{c}}\int_{\Omega}\gamma (d,\nabla d)\mathrm{d}\Omega = g_{\mathrm{c}}\int_{\Omega}\frac{1}{2l}\left(d^{2} + l^{2}\nabla d\cdot \nabla d\right)\mathrm{d}\Omega
\]

- 含义分解：
  - \[\Gamma_l(d)=\int_\Omega \gamma(d,\nabla d)\,\mathrm d\Omega\]：相场近似裂纹“面积/长度”泛函；
  - \[\gamma=\tfrac{1}{2l}(d^2+l^2|\nabla d|^2)\]：局部裂纹表面密度；
  - 乘 \[g_c\] 后得到真实裂纹耗散能 \[E_{\mathrm{crack}}\]。
- 必要推导（相场裂纹近似思想）：
  1. 经典 Griffith 裂纹能：\[E_{\mathrm{crack}}=g_c\,\mathcal H^{n-1}(\Gamma)\]；
  2. 以连续变量 \[d\] 近似离散裂纹集 \[\Gamma\]；
  3. 构造带内惩罚 \[d^2\]（控制损伤幅值）与梯度惩罚 \[l^2|\nabla d|^2\]（控制带宽）；
  4. 归一化因子 \[1/(2l)\] 使 \[l\to0\] 时可逼近 Griffith 裂纹能。
- 数值解释：
  - 若不可逆约束有效，卸载阶段 \[d\] 不回退，\[E_{\mathrm{crack}}\] 保持常数；
  - 若不约束，\[d\] 回退会导致 \[E_{\mathrm{crack}}\] 下降，违反热力学不可逆性。

### 公式 4.1-15（位移场投影梯度二范数）

\[
\| \mathrm{Proj}_{c}^{\pmb{u}}\|_{2}
\]

- 含义：约束投影后的位移方向残差信号。
- 越接近 0，表示位移子问题越接近 KKT 驻点。

### 公式 4.1-16（相场投影梯度二范数）

\[
\| \mathrm{Proj}_{c}^{d}\|_{2}
\]

- 含义：约束投影后的相场方向残差。
- 在卸载时常可降为接近 0（甚至 0），因为相场变量被下界卡住，几乎全为活动约束。

### 公式 4.1-17（二范数符号）

\[
l_2
\]

- 含义：欧氏范数，定义为 \[\|\mathbf v\|_2=\sqrt{\sum_i v_i^2}\]。

### 公式 4.1-18（收敛容差）

\[
10^{-6}
\]

- 含义：迭代停止阈值；当投影梯度范数与增量范数均降至该量级，并且活动集不再变化时判收敛。

\]

\[

\]

- 含义：加载路径参数，控制旋转角随步推进。

### 公式 4.4-8（扭转边界条件：y向位移）

\[
u_{y} = z \tan t
\]

- 含义：端面绕 \[x\] 轴旋转时，\[y\] 向位移与 \[z\] 成正比。
- 备注：该式在 `explain.md` 第4章中作为**规定的加载路径边界条件**给出，主要用于施加扭转型位移场。
- 说明（与标准刚体旋转几何的对比）：
  - 端面点 \[(y,z)\] 绕原点旋转角 \[t\] 后：
    \[
    y' = y\cos t + z\sin t,\quad
    z' = -y\sin t + z\cos t
    \]
  - 位移 \[u_y=y'-y=y(\cos t-1)+z\sin t\]。
  - `explain.md` 第4章中的边界条件是**直接规定**为 \[u_y=z\tan t\]（并与 \[u_z=-y\tan t\] 配对）作为加载路径定义。
  - 上面的刚体旋转表达仅用于帮助理解“旋转型位移场”的运动学背景，不作为对该边界条件的逐项代数推导。

### 公式 4.4-9（扭转边界条件：z向位移）

\[
u_{z} = -y \tan t
\]

- 含义：与上一式共同形成端面旋转位移场。
- 实现层面采用的是上面的 \[\tan t\] 形式（即文中给出的“精确施加式”）；仅在 \[|t|\ll 1\] 的解释中可用 \[\tan t\approx t\] 得到 \[u_y\approx zt,\ u_z\approx -yt\] 的线性化直观理解，这不是对原边界条件的替换。

## 第4章公式之间的整体逻辑串联（总解读）

1. **材料参数组**（\[\lambda,\mu,g_c\]）决定“弹性能储存能力 + 断裂耗能门槛”。
2. **相场参数组**（\[l,k,g(d)\]）决定裂纹带宽度、退化规律与数值稳定特征。
3. **网格准则**（\[h/l\] 或 \[(h/l)_{\max}\]）决定裂纹带是否被充分离散。
4. **不可逆箱体约束**（\[d^{(n)}\le d\le1\] 及四种状态式）保证裂纹不会“自愈”。
5. **裂纹能公式**（式33）把“损伤幅值 + 梯度正则”统一为可积能量项，与热力学一致性直接挂钩。
6. **收敛判据公式**（投影梯度范数与增量范数）把“方程残差收敛 + 约束活跃集稳定”统一到 L-BFGS-B 终止条件。
7. **3D 扭转位移边界公式**（\[u_y=z\tan t,\ u_z=-y\tan t\]）给出可控旋转加载，从而验证算法在复杂三维裂纹场中的能力。

### A. 从 Griffith 裂纹能到相场裂纹能（对应式33）

\[

\]

\[
\Downarrow\ \text{用连续相场 } d \text{ 近似离散裂纹面 } \Gamma

\[

\[
E_{\mathrm{crack}}=g_c\Gamma_l(d)
\]

- 解释：这一替换把“追踪裂纹几何拓扑”转化为“求解连续场 + 变分问题”，数值实现大幅简化。

- 这正是文中 \[\mathrm{L.U.B.}\] 状态的数学本质。

\[

u_y \approx zt,\quad u_z \approx -yt

- 含义：位移与到旋转轴距离成正比，符合刚体旋转的局部运动学结构。
